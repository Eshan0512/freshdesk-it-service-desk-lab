# freshdesk-it-service-desk-lab
# IT Service Desk Ticket Simulation (Freshdesk)

This repository documents a simulated L1 IT Service Desk environment using Freshdesk.
The project demonstrates incident intake, triage, troubleshooting, escalation awareness,
user communication, and resolution following ITIL-style workflows.

## Tools Used
- Freshdesk (ticketing system)
- Windows 10 Virtual Machine (VirtualBox)
- Windows diagnostic and account tools

## Tickets Documented
1. PC Boot Issue (Startup Repair)
2. Zebra Scanner Network Connectivity Issue
3. Email / Application Access Denied (Account Guidance)

All tickets were simulated in a controlled lab environment for learning and portfolio purposes.
No production systems or real user data were used.
