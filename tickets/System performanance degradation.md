## Ticket 3: System Performance Degradation – SSD Upgrade
![System Performance – Freshdesk Ticket](https://github.com/Eshan0512/freshdesk-it-service-desk-lab/blob/7c2f7371b2e40f88376527c4abb6a415e76b9657/Ticket_3.png)

**Type:** Incident  
**Priority:** Medium  

### Issue
A user reported that their laptop was running extremely slow, with long boot times and poor overall system performance affecting daily work.

### L1 Diagnosis
- Reviewed reported performance symptoms
- Identified mechanical HDD as the primary storage device
- Determined storage performance as the likely cause of slow boot and system responsiveness

### Resolution
Replaced the existing HDD with an SSD and reinstalled the operating system. Verified significantly improved boot times and overall system performance following the upgrade.

### Outcome
System performance was restored, and the user confirmed improved responsiveness and faster startup times.
